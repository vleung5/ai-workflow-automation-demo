📋 S3 Bucket Structure
The application uses the following S3 bucket folders for organizing files and results.
incoming/ New files waiting to be processed
processed/ Successfully processed files
failed/ Files that failed processing
archived/ Old or archived files
results/ Processing results in JSON format
🚀 Quick Start Guide
  1. Upload CSV files to: s3://<bucket>/incoming/
  2. Application automatically polls every 30 seconds
  3. Files are processed (validated, classified, summarized)
  4. Results saved to: results/<timestamp>/
  5. Processed files moved to: processed/<date>/
  6. Monitor progress in the Status and Jobs tabs
